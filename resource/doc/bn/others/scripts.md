# কাস্টম স্ক্রিপ্ট

কখনও কখনও আমাদের কিছু অস্থায়ী স্ক্রিপ্ট লিখতে হতে পারে, যেখানে আমরা ওয়েবম্যান এর মতো যেকোনো ক্লাস বা ইন্টারফেস কল করতে পারি, যেমন ডেটা ইম্পোর্ট, ডেটা আপডেট, পরিসংখ্যান ইত্যাদি কাজ সম্পন্ন করার জন্য। ওয়েবম্যান এ এটি অত্যন্ত সহজ, উদাহরণ:

**নতুন তৈরি `scripts/update.php`** (ডিরেক্টরি না থাকলে নিজে তৈরি করুন)
```php
<?php
require_once __DIR__ . '/../vendor/autoload.php';
require_once __DIR__ . '/../support/bootstrap.php';

use think\facade\Db;

$user = Db::table('user')->find(1);

var_dump($user);
```

অবশ্যই আমরা `webman/console` কাস্টম কমান্ড ব্যবহার করেও এই ধরনের অপারেশন সম্পাদন করতে পারি, দেখুন [কমান্ড লাইন](../plugin/console.md)।
