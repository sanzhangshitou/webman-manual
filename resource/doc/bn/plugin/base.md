# মৌলিক প্লাগইন

মৌলিক প্লাগইনগুলি সাধারণভাবে সাধারণ উপাদান, সাধারণত Composer দিয়ে ইনস্টল করা হয় এবং কোড vendor ডিরেক্টরির নিচে রাখা হয়। ইনস্টল করার সময় কাস্টম কনফিগারেশন (মিডলওয়্যার, প্রসেস, রাউট ইত্যাদি) `{মূল প্রকল্প}config/plugin` ডিরেক্টরিতে স্বয়ংক্রিয়ভাবে কপি করা যেতে পারে। webman এই ডিরেক্টরির কনফিগারেশন স্বয়ংক্রিয়ভাবে সনাক্ত করবে এবং প্রধান কনফিগারেশনের সাথে একত্রিত করবে, যাতে প্লাগইনগুলি webman-এর জীবনচক্রের যেকোনো পর্যায়ে হস্তক্ষেপ করতে পারে।

আরও তথ্যের জন্য [মৌলিক প্লাগইন তৈরি](create.md) দেখুন।
