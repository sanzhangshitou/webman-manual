# ডিরেক্টরি স্ট্রাকচার
```
.
├── app                           অ্যাপ্লিকেশন ডিরেক্টরি
│   ├── controller                কন্ট্রোলার ডিরেক্টরি
│   ├── model                     মডেল ডিরেক্টরি
│   ├── view                      ভিউ ডিরেক্টরি
│   ├── middleware                মিডলওয়্যার ডিরেক্টরি
│   │   └── StaticFile.php        অন্তর্নির্মিত স্ট্যাটিক ফাইল মিডলওয়্যার
│   ├── process                   কাস্টম প্রসেস ডিরেক্টরি
│   │   ├── Http.php              Http প্রসেস
│   │   └── Monitor.php           মনিটর প্রসেস
│   └── functions.php             ব্যবসায়িক কাস্টম ফাংশন এই ফাইলে লিখতে হবে
├── config                        কনফিগারেশন ডিরেক্টরি
│   ├── app.php                   অ্যাপ্লিকেশন কনফিগারেশন
│   ├── autoload.php              এখানে কনফিগার করা ফাইলগুলি স্বয়ংক্রিয়ভাবে লোড হবে
│   ├── bootstrap.php             প্রসেস শুরু হলে onWorkerStart এ চালানো ক্যালব্যাক কনফিগারেশন
│   ├── container.php             কন্টেনার কনফিগারেশন
│   ├── dependence.php            কন্টেনার নির্ভরতা কনফিগারেশন
│   ├── database.php             ডাটাবেস কনফিগারেশন
│   ├── exception.php             এক্সেপশন কনফিগারেশন
│   ├── log.php                   লগ কনফিগারেশন
│   ├── middleware.php            মিডলওয়্যার কনফিগারেশন
│   ├── process.php               কাস্টম প্রসেস কনফিগারেশন
│   ├── redis.php                 রেডিস কনফিগারেশন
│   ├── route.php                 রাউট কনফিগারেশন
│   ├── server.php                পোর্ট, প্রসেস সংখ্যা ইত্যাদি সার্ভার কনফিগারেশন
│   ├── view.php                  ভিউ কনফিগারেশন
│   ├── static.php                স্ট্যাটিক ফাইল সুইচ এবং মিডলওয়্যার কনফিগারেশন
│   ├── translation.php           বহুভাষিক কনফিগারেশন
│   └── session.php               সেশন কনফিগারেশন
├── public                        স্ট্যাটিক রিসোর্স ডিরেক্টরি
├── runtime                       অ্যাপ্লিকেশনের রানটাইম ডিরেক্টরি, লিখন অনুমতি প্রয়োজন
├── start.php                     সেবা স্টার্ট ফাইল
├── vendor                        Composer দ্বারা ইনস্টল করা তৃতীয় পক্ষ লাইব্রেরি ডিরেক্টরি
└── support                       লাইব্রেরি অভিযোজন (তৃতীয় পক্ষ লাইব্রেরি সহ)
    ├── Request.php               অনুরোধ ক্লাস
    ├── Response.php              প্রতিক্রিয়া ক্লাস
    ├── Setup.php                 ইন্সটলেশন উইজার্ড স্ক্রিপ্ট
    └── bootstrap.php             প্রসেস স্টার্টের পর ইনিশিয়ালাইজেশন স্ক্রিপ্ট
```
