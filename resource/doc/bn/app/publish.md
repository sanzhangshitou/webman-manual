# প্রকাশনা

ডেভেলপাররা তাদের নিজস্ব কপিরাইট প্লাগইন অ্যাপ মার্কেটে আপলোড করতে পারেন, অন্যদের সাহায্য করার পাশাপাশি নিজেরাও লাভবান হতে পারেন।
প্রকাশনা ঠিকানা: https://www.workerman.net/app/create

দ্রষ্টব্য: প্রকাশনার সময় পূরণ করা **সংস্করণ নম্বর**টি `plugin/{plugin}/config/app.php`-এ সংস্করণ নম্বরের সাথে মিলতে হবে।

## তালিকাভুক্তকরণ সীমাবদ্ধতা
**নিম্নলিখিত ধরণের অ্যাপ্লিকেশন তালিকাভুক্তকরণের জন্য সমর্থিত নয়:**

**আর্থিক** অন্তর্ভুক্ত কিন্তু সীমাবদ্ধ নয়: সমষ্টিগত পেমেন্ট, ক্যাশব্যাক, ঋণ, নিলাম, ডিজিটাল মুদ্রা, ব্লকচেইন  
**বোর্ড ও কার্ড গেম** অন্তর্ভুক্ত কিন্তু সীমাবদ্ধ নয়: লটারি, ধনরত্ন খোঁজা, NiuNiu, প্রিপেইড কার্ড ইত্যাদি  
**ডেটা সংগ্রহ** অন্তর্ভুক্ত কিন্তু সীমাবদ্ধ নয়: উপন্যাস, কমিক্স, সিনেমা, টিভি শো, সম্পদ ভাগাভাগি ইত্যাদি  
**বিদেশী অ্যাপ্লিকেশন** অন্তর্ভুক্ত কিন্তু সীমাবদ্ধ নয়: ক্লাউড হোস্টিং, VPN ইত্যাদি  
**চ্যাট এবং গ্রাহক সেবা** বর্তমানে তালিকাভুক্তকরণের জন্য সমর্থিত নয়।
