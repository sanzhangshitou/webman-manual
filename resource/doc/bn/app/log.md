# লগ
লগ ক্লাসের ব্যবহার ডাটাবেস ব্যবহারের মতই
```php
use support\Log;
Log::channel('plugin.admin.default')->info('test');
```

মূল প্রকল্পের লগ কনফিগারেশন পুনরায় ব্যবহার করতে চাইলে সরাসরি ব্যবহার করুন:
```php
use support\Log;
Log::info('লগ বিষয়বস্তু');
// ধরে নিন যে মূল প্রকল্পে test নামের একটি লগ কনফিগারেশন রয়েছে
Log::channel('test')->info('লগ বিষয়বস্তু');
```
