# রাউট কনফিগারেশন ফাইল
প্লাগইনের রাউট কনফিগারেশন ফাইলটি অবস্থিত `plugin/প্লাগইন_নাম/config/route.php` এ

## ডিফল্ট রাউট
অ্যাপ্লিকেশন প্লাগইনের URL ঠিকানার পথগুলি 모두 `/app` দিয়ে শুরু হয়, উদাহরণস্বরূপ `plugin\foo\app\controller\UserController` এর URL ঠিকানা হল `http://127.0.0.1:8787/app/foo/user`

## ডিফল্ট রাউট নিষ্ক্রিয় করুন
কোনো অ্যাপ্লিকেশন প্লাগইনের ডিফল্ট রাউট নিষ্ক্রিয় করতে চাইলে, রাউট কনফিগারেশনে নিম্নরূপ সেট করুন:
```php
Route::disableDefaultRoute('foo');
```

## 404 কলব্যাক পরিচালনা
কোনো অ্যাপ্লিকেশন প্লাগইনের জন্য fallback সেট করতে চাইলে, দ্বিতীয় প্যারামিটার হিসেবে প্লাগইনের নাম পাস করতে হবে। উদাহরণ:
```php
Route::fallback(function(){
    return redirect('/');
}, 'foo');
```
