# Upgrading Method

`composer require workerman/webman-framework ^1.4.3 && composer require webman/console ^1.0.27 && php webman install`

> **Note**
> Alibaba Cloud's Composer proxy has stopped syncing data from the official Composer repository. You cannot upgrade to the latest webman using the Alibaba Cloud Composer proxy. Please run the following command to switch back to the official Composer data source: `composer config -g --unset repos.packagist`
