# Integrate webman-admin

[webman-admin](https://www.workerman.net/plugin/82) is the official management backend developed by webman. It supports permission settings, menu generation (one-click CRUD), and comes with a user table to facilitate sharing user data across application plugins.

## Benefits of integrating webman-admin

* Provides a unified backend entry for various application plugins, improving user experience
* Reduces backend development time for application plugins and lowers maintenance costs
* The management backend includes plugin management functionality for centralized search and installation of plugins
