# السكريبتات المخصصة

أحياناً نحتاج إلى كتابة بعض السكريبتات المؤقتة، يمكننا في هذه السكريبتات استدعاء أي كلاس أو واجهة تماماً كما يفعل webman، لإكمال عمليات مثل استيراد البيانات، تحديث البيانات، والإحصائيات وما إلى ذلك. هذا أمر سهل جداً في webman، على سبيل المثال:

**إنشاء ملف جديد `scripts/update.php`** (إذا لم يكن المجلد موجوداً، يرجى إنشاؤه يدوياً)
```php
<?php
require_once __DIR__ . '/../vendor/autoload.php';
require_once __DIR__ . '/../support/bootstrap.php';

use think\facade\Db;

$user = Db::table('user')->find(1);

var_dump($user);
```

بطبيعة الحال، يمكننا أيضاً استخدام أوامر `webman/console` المخصصة لإكمال مثل هذه العمليات، راجع [سطر الأوامر](../plugin/console.md)
