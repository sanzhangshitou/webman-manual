# طريقة الترقية

`composer require workerman/webman-framework ^1.4.3 && composer require webman/console ^1.0.27 && php webman install`

> **ملاحظة**
> توقف وكيل Composer من Alibaba Cloud عن مزامنة البيانات من المصدر الرسمي لـ Composer. لا يمكن الترقية إلى أحدث إصدار من webman باستخدام وكيل Alibaba Cloud. يرجى تنفيذ الأمر التالي للعودة إلى استخدام المصدر الرسمي لبيانات Composer: `composer config -g --unset repos.packagist`
