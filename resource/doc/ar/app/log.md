# السجل
استخدام فئة السجل يشبه استخدام قاعدة البيانات
```php
use support\Log;
Log::channel('plugin.admin.default')->info('test');
```

إذا أردت إعادة استخدام إعدادات السجل الخاصة بالمشروع الرئيسي، استخدم مباشرة:
```php
use support\Log;
Log::info('محتوى السجل');
// بافتراض أن المشروع الرئيسي يحتوي على إعداد سجل باسم test
Log::channel('test')->info('محتوى السجل');
```
