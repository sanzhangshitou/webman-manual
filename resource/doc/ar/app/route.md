# ملف تكوين المسارات
ملف تكوين المسارات للإضافات موجود في `plugin/اسم_الإضافة/config/route.php`

## المسار الافتراضي
جميع مسارات عناوين URL لتطبيقات الإضافات تبدأ بـ `/app`، فعلى سبيل المثال عنوان URL لـ `plugin\foo\app\controller\UserController` هو `http://127.0.0.1:8787/app/foo/user`

## تعطيل المسار الافتراضي
لتعطيل المسار الافتراضي لتطبيق إضافة معيّن، أضف التالي في تكوين المسارات:
```php
Route::disableDefaultRoute('foo');
```

## معالجة استدعاء 404
لتعيين fallback لتطبيق إضافة معيّن، يجب تمرير اسم الإضافة كمعامل ثانٍ. مثال:
```php
Route::fallback(function(){
    return redirect('/');
}, 'foo');
```
