# تكامل webman-admin

[webman-admin](https://www.workerman.net/plugin/82) لوحة إدارة رسمية طوّرها فريق webman. يدعم إعداد الصلاحيات وإنشاء القوائم (CRUD بنقرة واحدة)، ويأتي مع جدول المستخدمين لتسهيل تشارك بيانات المستخدمين بين مختلف ملحقات التطبيقات.

## مزايا تكامل webman-admin

* توفير نقطة دخول موحدة للوحة الإدارة لجميع ملحقات التطبيقات، مما يحسّن تجربة المستخدم
* تقليل وقت تطوير لوحة الإدارة للملحقات وخفض تكاليف الصيانة
* لوحة الإدارة مزودة بوظيفة إدارة الملحقات، مما يسهل البحث المركزي وتركيب الملحقات
