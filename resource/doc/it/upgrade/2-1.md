# Guida all'aggiornamento 2.1

## Funzionalità e modifiche

* Richiede `PHP>=8.1`

* Dipende da `workerman/workerman:^5.1`, supporto coroutine. Vedi [webman Coroutine](../coroutine/coroutine.md)

* Aggiunto pool di connessioni per `webman/redis:^2.1` `webman/cache:^2.1` `webman/database:^2.1` `webman/think-orm:^2.1` `webman/think-cache:^2.1`, supporto automatico per ambienti con e senza coroutine.

**Eseguire un backup prima dell'aggiornamento, quindi eseguire i seguenti comandi**
```
composer config -g --unset repos.packagist
composer require -W workerman/workerman:~5.1 workerman/webman-framework:~2.1
```

**Aggiornamento database**

Se si utilizzano i seguenti componenti, eseguire i comandi corrispondenti:

* `illuminate/database` di Laravel
```
composer require -W webman/database:~2.1
```

* `illuminate/redis` di Laravel
```
composer require -W webman/redis:~2.1
```

* `illuminate/cache` di Laravel
```
composer require -W webman/cache:~2.1
```

* `topthink/think-orm` di ThinkPHP
```
composer require -W webman/think-orm:~2.1
```

* `topthink/think-cache` di ThinkPHP
```
composer require -W webman/think-cache:~2.1
```

* webman/redis-queue
```
composer require -W webman/redis-queue:~2.1
```

* webman/log
```
composer require -W webman/log:~2.1
```
