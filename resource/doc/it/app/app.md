# Plugin dell'applicazione

## Concetto
Costruisci la tua applicazione rapidamente, come se stessi assemblando blocchi.

## Introduzione
Un plugin dell'applicazione è in realtà un'applicazione completa che può essere installata come plugin nel progetto principale, permettendo al progetto principale di acquisire rapidamente funzionalità di moduli specifici.
Ad esempio, se il progetto principale ha bisogno di un sistema Q&A, puoi installare un plugin dell'applicazione Q&A; se ha bisogno di un sistema e-commerce, puoi installare un plugin dell'applicazione e-commerce. Tutti i plugin sono indipendenti l'uno dall'altro, plug-and-play, e possono essere rimossi con un clic quando non sono più necessari.

## Impatto
I plugin dell'applicazione migliorano notevolmente la riutilizzabilità dei progetti webman. Qualsiasi sviluppatore può trasformare il proprio progetto in un plugin e offrirlo per l'installazione in altri progetti, beneficiando quei progetti e ottenendo profitti per sé stesso.

## Ottenere profitti
webman fornisce un [marketplace ufficiale delle applicazioni](https://www.workerman.net/apps) dove gli sviluppatori possono inviare le applicazioni che hanno sviluppato e guadagnare da esse.
