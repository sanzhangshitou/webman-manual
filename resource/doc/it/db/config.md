# Configurazione database (stile Laravel)
webman/database supporta i seguenti database e versioni:

- MySQL 5.6+
- PostgreSQL 9.4+
- SQLite 3.8.8+
- SQL Server 2017+

Il file di configurazione si trova in `config/database.php`.

```php
return [
    // Database predefinito
    'default' => 'mysql',
    // Varie configurazioni database
    'connections' => [

        'mysql' => [
            'driver'      => 'mysql',
            'host'        => '127.0.0.1',
            'port'        => 3306,
            'database'    => 'webman',
            'username'    => 'webman',
            'password'    => '',
            'unix_socket' => '',
            'charset'     => 'utf8',
            'collation'   => 'utf8_unicode_ci',
            'prefix'      => '',
            'strict'      => true,
            'engine'      => null,
            'pool' => [ // Configurazione pool connessioni, solo con driver swoole/swow
                'max_connections' => 5, // Numero massimo di connessioni
                'min_connections' => 1, // Numero minimo di connessioni
                'wait_timeout' => 3,    // Tempo massimo di attesa per ottenere una connessione, eccezione al timeout
                'idle_timeout' => 60,   // Tempo massimo di inattività delle connessioni nel pool, recupero fino a min_connections
                'heartbeat_interval' => 50, // Intervallo heartbeat in secondi, consigliato meno di 60
            ],
         ],
         
         'sqlite' => [
             'driver'   => 'sqlite',
             'database' => '',
             'prefix'   => '',
             'pool' => [
                'max_connections' => 5,
                'min_connections' => 1,

                'idle_timeout' => 60,

         'pgsql' => [
             'driver'   => 'pgsql',

             'username' => 'webman',

             'charset'  => 'utf8',
             'prefix'   => '',
             'schema'   => 'public',
             'sslmode'  => 'prefer',
             'pool' => [
                'max_connections' => 5,
                'min_connections' => 1,
```

## Usare più database
Usare `Db::connection('nome_config')` per scegliere il database, dove `nome_config` è la `key` corrispondente nel file `config/database.php`.

Esempio con la seguente configurazione:

```php
 return [
     'default' => 'mysql',
     'connections' => [

         'mysql' => [
             'driver'      => 'mysql',
             'host'        =>   '127.0.0.1',
             'port'        => 3306,
             'database'    => 'webman',
             'username'    => 'webman',
             'password'    => '',
             'unix_socket' =>  '',
             'charset'     => 'utf8',
             'collation'   => 'utf8_unicode_ci',
             'prefix'      => '',
             'strict'      => true,
             'engine'      => null,
         ],
         
         'mysql2' => [
              'driver'      => 'mysql',
              'host'        => '127.0.0.1',
              'port'        => 3306,
              'database'    => 'webman2',
              'username'    => 'webman2',
              'password'    => '',
              'unix_socket' => '',
              'charset'     => 'utf8',
              'collation'   => 'utf8_unicode_ci',
              'prefix'      => '',
              'strict'      => true,
              'engine'      => null,
         ],
         'pgsql' => [
              'driver'   => 'pgsql',
              'host'     => '127.0.0.1',
              'port'     =>  5432,
              'database' => 'webman',
              'username' =>  'webman',
              'password' => '',
              'charset'  => 'utf8',
              'prefix'   => '',
              'schema'   => 'public',
              'sslmode'  => 'prefer',
          ],
 ];
```

Per cambiare database:
```php
// Usa database predefinito, equivalente a Db::connection('mysql')->table('users')->where('name', 'John')->first();
$users = Db::table('users')->where('name', 'John')->first(); 
// Usa mysql2
$users = Db::connection('mysql2')->table('users')->where('name', 'John')->first();
// Usa pgsql
$users = Db::connection('pgsql')->table('users')->where('name', 'John')->first();
```
