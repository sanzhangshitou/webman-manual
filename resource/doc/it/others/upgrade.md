# Metodo di aggiornamento

`composer require workerman/webman-framework ^1.4.3 && composer require webman/console ^1.0.27 && php webman install`

> **Nota**
> Il proxy Composer di Alibaba Cloud ha interrotto la sincronizzazione dei dati dal repository ufficiale di Composer. Non è possibile aggiornare webman all'ultima versione tramite il proxy Alibaba Cloud. Eseguire il comando seguente per tornare a utilizzare la sorgente dati ufficiale di Composer: `composer config -g --unset repos.packagist`
