# Upgrademethode

`composer require workerman/webman-framework ^1.4.3 && composer require webman/console ^1.0.27 && php webman install`

> **Hinweis**
> Der Composer-Proxy von Alibaba Cloud hat die Datensynchronisation mit dem offiziellen Composer-Repository eingestellt. Eine Aktualisierung auf die neueste webman-Version über den Alibaba Cloud Composer-Proxy ist derzeit nicht möglich. Bitte führen Sie folgenden Befehl aus, um wieder die offizielle Composer-Datenquelle zu verwenden: `composer config -g --unset repos.packagist`
