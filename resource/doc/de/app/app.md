# Anwendungs-Plugins

## Konzept
Baue deine eigene Anwendung schnell auf – wie beim Zusammenstecken von Bausteinen.

## Einführung
Ein Anwendungs-Plugin ist tatsächlich eine vollständige Anwendung, die als Plugin in das Hauptprojekt installiert werden kann, sodass das Hauptprojekt schnell bestimmte Modulfunktionen erhält.
Benötigt das Hauptprojekt beispielsweise ein Q&A-System, kann ein Q&A-Anwendungs-Plugin installiert werden; benötigt es ein E-Commerce-System, kann ein entsprechendes Anwendungs-Plugin installiert werden. Alle Plugins beeinflussen sich gegenseitig nicht, sind plug-and-play und können bei Bedarf mit einem Klick entfernt werden.

## Auswirkungen
Anwendungs-Plugins steigern die Wiederverwendbarkeit von webman-Projekten erheblich. Jeder Entwickler kann sein Projekt in ein Plugin umwandeln und anderen Projekten zur Installation und Nutzung anbieten – anderen Projekten zugutekommend und selbst Einnahmen erzielend.

## Einnahmen erzielen
webman bietet einen offiziellen [Anwendungsmarkt](https://www.workerman.net/apps), auf dem Entwickler ihre entwickelten Anwendungen einreichen und daraus Einnahmen erzielen können.
