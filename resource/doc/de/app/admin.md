# webman-admin integrieren

[webman-admin](https://www.workerman.net/plugin/82) ist das offizielle Verwaltungs-Backend von webman. Es unterstützt Berechtigungseinstellungen, Menügenerierung (CRUD per Klick) und eine Benutzertabelle für die gemeinsame Nutzung von Benutzerdaten zwischen Anwendungs-Plugins.

## Vorteile der webman-admin-Integration

* Bietet eine einheitliche Backend-Einstiegsstelle für verschiedene Anwendungs-Plugins und verbessert die Benutzererfahrung
* Verkürzt die Backend-Entwicklungszeit für Anwendungs-Plugins und senkt die Wartungskosten
* Das Verwaltungs-Backend enthält eine Plugin-Verwaltung für zentrale Suche und Installation von Plugins
