# 接入 webman-admin

[webman-admin](https://www.workerman.net/plugin/82) 是 webman 官方開發的管理後台。支援權限設置、菜單生成（一鍵 CRUD），自帶用戶表，方便各應用插件共享用戶數據。

## 接入 webman-admin 好處

* 為各應用插件提供統一的後台入口，提升用戶體驗
* 減少應用插件後台開發時間，降低維護成本
* 管理後台自帶插件管理功能，方便插件的集中檢索與安裝
