# Intégrer webman-admin

[webman-admin](https://www.workerman.net/plugin/82) est le panneau d'administration officiel développé par webman. Il prend en charge la configuration des autorisations, la génération des menus (CRUD en un clic) et inclut une table utilisateurs pour faciliter le partage des données entre les plugins d'applications.

## Avantages de l'intégration de webman-admin

* Offre une entrée unique au panneau d'administration pour les différents plugins d'applications, améliorant l'expérience utilisateur
* Réduit le temps de développement du panneau pour les plugins et diminue les coûts de maintenance
* Le panneau comprend une gestion des plugins pour la recherche et l'installation centralisées
