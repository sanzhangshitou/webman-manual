# Méthode de mise à niveau

`composer require workerman/webman-framework ^1.4.3 && composer require webman/console ^1.0.27 && php webman install`

> **Remarque**
> Le proxy Composer d'Alibaba Cloud a cessé de synchroniser les données depuis le dépôt officiel de Composer. Il n'est pas possible de mettre à jour webman vers la dernière version en utilisant le proxy Alibaba Cloud. Veuillez exécuter la commande suivante pour revenir à la source de données officielle de Composer : `composer config -g --unset repos.packagist`
