# अपग्रेड विधि

`composer require workerman/webman-framework ^1.4.3 && composer require webman/console ^1.0.27 && php webman install`

> **ध्यान दें**
> Alibaba Cloud का Composer प्रॉक्सी Composer के आधिकारिक स्रोत से डेटा सिंक करना बंद कर चुका है। Alibaba Cloud Composer प्रॉक्सी का उपयोग करके नवीनतम webman तक अपग्रेड करना संभव नहीं है। Composer के आधिकारिक डेटा स्रोत का उपयोग फिर से शुरू करने के लिए निम्न कमांड चलाएँ: `composer config -g --unset repos.packagist`
