# स्थापना

ऐप प्लगइन स्थापित करने के दो तरीके हैं:

## प्लगइन बाज़ार से स्थापित करें
[आधिकारिक प्रबंधन पैनल webman-admin](https://www.workerman.net/plugin/82) पर जाएँ, ऐप प्लगइन पेज खोलें और संबंधित ऐप प्लगइन स्थापित करने के लिए इंस्टॉल बटन पर क्लिक करें।

## सोर्स पैकेज से स्थापित करें
ऐप बाज़ार से ऐप प्लगइन का आर्काइव डाउनलोड करें, अनज़िप करें और अनज़िप किए गए फ़ोल्डर को `{मुख्य प्रोजेक्ट}/plugin/` में अपलोड करें (plugin फ़ोल्डर न हो तो मैनुअल बनाएँ)। `php webman app-plugin:install प्लगइन_नाम` चलाकर स्थापना पूरी करें।

उदाहरण: अगर डाउनलोड किया गया आर्काइव ai.zip है, तो `{मुख्य प्रोजेक्ट}/plugin/ai` में अनज़िप करें और `php webman app-plugin:install ai` चलाकर स्थापना पूरी करें।


# अनइंस्टॉल

ऐप प्लगइन अनइंस्टॉल करने के भी दो तरीके हैं:

## प्लगइन बाज़ार से अनइंस्टॉल करें
[आधिकारिक प्रबंधन पैनल webman-admin](https://www.workerman.net/plugin/82) पर जाएँ, ऐप प्लगइन पेज खोलें और संबंधित ऐप प्लगइन अनइंस्टॉल करने के लिए अनइंस्टॉल बटन पर क्लिक करें।

## सोर्स पैकेज से अनइंस्टॉल करें
`php webman app-plugin:uninstall प्लगइन_नाम` चलाकर अनइंस्टॉल पूरा करें। उसके बाद `{मुख्य प्रोजेक्ट}/plugin/` में संबंधित प्लगइन फ़ोल्डर हाथ से हटा दें।
