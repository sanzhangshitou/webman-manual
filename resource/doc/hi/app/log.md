# लॉग
लॉग कक्षा का उपयोग डेटाबेस के उपयोग के समान है
```php
use support\Log;
Log::channel('plugin.admin.default')->info('test');
```

यदि मुख्य परियोजना की लॉग कॉन्फ़िगरेशन का पुन: उपयोग करना हो तो सीधे इस प्रकार उपयोग करें:
```php
use support\Log;
Log::info('लॉग सामग्री');
// मान लें कि मुख्य परियोजना में test नाम की लॉग कॉन्फ़िगरेशन है
Log::channel('test')->info('लॉग सामग्री');
```
