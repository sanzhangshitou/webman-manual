# रूट कॉन्फ़िगरेशन फ़ाइल
प्लगइन की रूट कॉन्फ़िगरेशन फ़ाइल `plugin/प्लगइन_नाम/config/route.php` में स्थित है।

## डिफ़ॉल्ट रूट
एप्लिकेशन प्लगइन के सभी URL पथ `/app` से शुरू होते हैं, उदाहरण के लिए `plugin\foo\app\controller\UserController` का URL है `http://127.0.0.1:8787/app/foo/user`

## डिफ़ॉल्ट रूट अक्षम करें
किसी एप्लिकेशन प्लगइन का डिफ़ॉल्ट रूट अक्षम करने के लिए, रूट कॉन्फ़िगरेशन में निम्नलिखित जोड़ें:
```php
Route::disableDefaultRoute('foo');
```

## 404 कॉलबैक संभालें
किसी एप्लिकेशन प्लगइन के लिए fallback सेट करने के लिए, दूसरे पैरामीटर के रूप में प्लगइन का नाम पास करना होगा। उदाहरण:
```php
Route::fallback(function(){
    return redirect('/');
}, 'foo');
```
