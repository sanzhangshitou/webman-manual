# インストール

アプリケーションプラグインのインストールには2つの方法があります：

## プラグインマーケットからインストール
[公式管理画面 webman-admin](https://www.workerman.net/plugin/82) のアプリケーションプラグインのページにアクセスし、インストールボタンをクリックして対応するアプリケーションプラグインをインストールします。

## ソースパッケージからインストール
アプリケーションマーケットからアプリケーションプラグインの圧縮パッケージをダウンロードし、解凍して `{メインプロジェクト}/plugin/` にアップロードします（plugin ディレクトリがない場合は手動で作成してください）。`php webman app-plugin:install プラグイン名` を実行してインストールを完了します。

例えば、ダウンロードした圧縮パッケージ名が ai.zip の場合、`{メインプロジェクト}/plugin/ai` に解凍し、`php webman app-plugin:install ai` を実行してインストールを完了します。


# アンインストール

アプリケーションプラグインのアンインストールにも2つの方法があります：

## プラグインマーケットからアンインストール
[公式管理画面 webman-admin](https://www.workerman.net/plugin/82) のアプリケーションプラグインのページにアクセスし、アンインストールボタンをクリックして対応するアプリケーションプラグインをアンインストールします。

## ソースパッケージからアンインストール
`php webman app-plugin:uninstall プラグイン名` を実行してアンインストールを完了し、その後に `{メインプロジェクト}/plugin/` 内の対応するプラグインディレクトリを手動で削除してください。
