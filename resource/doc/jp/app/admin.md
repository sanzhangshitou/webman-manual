# webman-adminの導入

[webman-admin](https://www.workerman.net/plugin/82) はwebman公式が開発した管理バックエンドです。権限設定、メニュー生成（ワンクリックCRUD）をサポートし、ユーザーテーブルを内蔵しており、各アプリケーションプラグイン間でのユーザーデータ共有を容易にします。

## webman-admin導入のメリット

* 各アプリケーションプラグインに統一されたバックエンドへの入り口を提供し、ユーザー体験を向上させる
* アプリケーションプラグインのバックエンド開発時間を短縮し、保守コストを削減する
* 管理バックエンドにプラグイン管理機能を内蔵し、プラグインの集中検索・インストールを容易にする
