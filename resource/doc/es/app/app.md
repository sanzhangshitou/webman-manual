# Plugins de aplicación

## Concepto
Construye tu propia aplicación rápidamente, como si montaras bloques.

## Introducción
Un plugin de aplicación es en realidad una aplicación completa que puede instalarse como plugin en el proyecto principal, permitiendo que este adquiera rápidamente funcionalidades de módulos específicos.
Por ejemplo, si el proyecto principal necesita un sistema de preguntas y respuestas, puede instalar un plugin de aplicación Q&A; si necesita un sistema de comercio electrónico, puede instalar un plugin de aplicación de comercio electrónico. Todos los plugins son independientes entre sí, plug-and-play, y pueden eliminarse con un solo clic cuando ya no sean necesarios.

## Impacto
Los plugins de aplicación mejoran enormemente la reutilización de los proyectos webman. Cualquier desarrollador puede convertir su proyecto en un plugin y ofrecerlo para su instalación en otros proyectos, beneficiando a esos proyectos y obteniendo ingresos para sí mismo.

## Obtener ingresos
webman proporciona un [mercado de aplicaciones](https://www.workerman.net/apps) oficial donde los desarrolladores pueden enviar sus aplicaciones desarrolladas y obtener ingresos de ellas.
