# Método de actualización

`composer require workerman/webman-framework ^1.4.3 && composer require webman/console ^1.0.27 && php webman install`

> **Nota**
> El proxy de Composer de Alibaba Cloud ha dejado de sincronizar datos desde el repositorio oficial de Composer. No es posible actualizar a la última versión de webman usando el proxy de Alibaba Cloud. Use el siguiente comando para volver a utilizar la fuente de datos oficial de Composer: `composer config -g --unset repos.packagist`
