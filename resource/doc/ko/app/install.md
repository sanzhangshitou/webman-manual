# 설치

애플리케이션 플러그인 설치 방법에는 두 가지가 있습니다.

## 플러그인 시장에서 설치
[공식 관리자 대시보드 webman-admin](https://www.workerman.net/plugin/82) 의 애플리케이션 플러그인 페이지로 이동하여 설치 버튼을 클릭해 해당 애플리케이션 플러그인을 설치합니다.

## 소스 패키지에서 설치
애플리케이션 시장에서 애플리케이션 플러그인 압축 파일을 다운로드한 뒤 압축을 풀고, 풀린 디렉터리를 `{메인 프로젝트}/plugin/` 에 업로드합니다(plugin 디렉터리가 없으면 수동으로 생성). `php webman app-plugin:install 플러그인명` 을 실행해 설치를 완료합니다.

예: 다운로드한 압축 파일 이름이 ai.zip인 경우, `{메인 프로젝트}/plugin/ai` 에 압축을 풀고 `php webman app-plugin:install ai` 를 실행해 설치를 완료합니다.


# 제거

애플리케이션 플러그인 제거 방법에도 두 가지가 있습니다.

## 플러그인 시장에서 제거
[공식 관리자 대시보드 webman-admin](https://www.workerman.net/plugin/82) 의 애플리케이션 플러그인 페이지로 이동하여 제거 버튼을 클릭해 해당 애플리케이션 플러그인을 제거합니다.

## 소스 패키지에서 제거
`php webman app-plugin:uninstall 플러그인명` 을 실행해 제거를 완료한 뒤, `{메인 프로젝트}/plugin/` 디렉터리에서 해당 플러그인 디렉터리를 수동으로 삭제합니다.
