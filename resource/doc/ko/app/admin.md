# webman-admin 연동

[webman-admin](https://www.workerman.net/plugin/82)은 webman 공식에서 개발한 관리 백엔드입니다. 권한 설정, 메뉴 생성(원클릭 CRUD)을 지원하며 사용자 테이블이 내장되어 있어 각 애플리케이션 플러그인 간 사용자 데이터 공유가 편리합니다.

## webman-admin 연동 장점

* 각 애플리케이션 플러그인에 통합된 백엔드 진입점을 제공하여 사용자 경험을 향상시킵니다
* 애플리케이션 플러그인의 백엔드 개발 시간을 단축하고 유지보수 비용을 낮춥니다
* 관리 백엔드에 플러그인 관리 기능이 내장되어 플러그인 중앙 검색 및 설치가 편리합니다
