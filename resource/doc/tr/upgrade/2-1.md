# 2.1 Yükseltme Kılavuzu

## Özellikler ve Değişiklikler

* `PHP>=8.1` gerekir

* `workerman/workerman:^5.1` bağımlılığı, coroutine desteği. Bkz. [webman Coroutine](../coroutine/coroutine.md)

* `webman/redis:^2.1` `webman/cache:^2.1` `webman/database:^2.1` `webman/think-orm:^2.1` `webman/think-cache:^2.1` için bağlantı havuzu eklendi, coroutine ve coroutine dışı ortamlar için otomatik destek.

**Yükseltmeden önce yedek alın ve aşağıdaki komutları çalıştırın**
```
composer config -g --unset repos.packagist
composer require -W workerman/workerman:~5.1 workerman/webman-framework:~2.1
```

**Veritabanı yükseltmesi**

Aşağıdaki bileşenleri kullanıyorsanız karşılık gelen komutları çalıştırın:

* Laravel'in `illuminate/database`
```
composer require -W webman/database:~2.1
```

* Laravel'in `illuminate/redis`
```
composer require -W webman/redis:~2.1
```

* Laravel'in `illuminate/cache`
```
composer require -W webman/cache:~2.1
```

* ThinkPHP'nin `topthink/think-orm`
```
composer require -W webman/think-orm:~2.1
```

* ThinkPHP'nin `topthink/think-cache`
```
composer require -W webman/think-cache:~2.1
```

* webman/redis-queue
```
composer require -W webman/redis-queue:~2.1
```

* webman/log
```
composer require -W webman/log:~2.1
```
