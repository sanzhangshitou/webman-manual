# Güncelleme Yöntemi

`composer require workerman/webman-framework ^1.4.3 && composer require webman/console ^1.0.27 && php webman install`

> **Not**
> Alibaba Cloud Composer proxy'si Composer resmi deposundan veri senkronizasyonunu durdurduğu için, Alibaba Cloud Composer proxy kullanılarak en son webman sürümüne güncelleme yapılamaz. Composer resmi veri kaynağını kullanmaya devam etmek için şu komutu çalıştırın: `composer config -g --unset repos.packagist`
