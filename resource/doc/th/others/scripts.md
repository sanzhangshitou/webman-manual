# สคริปต์ที่กำหนดเอง

บางครั้งเราต้องเขียนสคริปต์ชั่วคราว ซึ่งในสคริปต์เหล่านี้เราสามารถเรียกใช้คลาสหรืออินเทอร์เฟซใดๆ ได้เหมือนกับ webman เพื่อทำงานเช่น การนำเข้าข้อมูล การอัปเดตข้อมูล สถิติ เป็นต้น ใน webman ทำได้ง่ายมาก ตัวอย่างเช่น:

**สร้าง `scripts/update.php`** (หากยังไม่มีโฟลเดอร์ให้สร้างเอง)
```php
<?php
require_once __DIR__ . '/../vendor/autoload.php';
require_once __DIR__ . '/../support/bootstrap.php';

use think\facade\Db;

$user = Db::table('user')->find(1);

var_dump($user);
```

แน่นอนว่าเรายังสามารถใช้คำสั่งที่กำหนดเองของ `webman/console` เพื่อดำเนินการเช่นนี้ได้ ดูที่ [คอมมานด์ไลน์](../plugin/console.md)
