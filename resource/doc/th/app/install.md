# การติดตั้ง

การติดตั้งปลั๊กอินแอปพลิเคชันมี 2 วิธี:

## ติดตั้งจากตลาดปลั๊กอิน
เข้า [แผงจัดการอย่างเป็นทางการ webman-admin](https://www.workerman.net/plugin/82) ไปที่หน้าปลั๊กอินแอปพลิเคชัน แล้วกดปุ่มติดตั้งเพื่อติดตั้งปลั๊กอินที่ต้องการ

## ติดตั้งจากแพ็กเกจซอร์สโค้ด
ดาวน์โหลดไฟล์บีบอัดปลั๊กอินจากตลาดแอปพลิเคชัน แกะไฟล์แล้วอัปโหลดโฟลเดอร์ที่แตกแล้วไปยัง `{โปรเจกต์หลัก}/plugin/` (ถ้าไม่มีโฟลเดอร์ plugin ให้สร้างด้วยตนเอง) จากนั้นรัน `php webman app-plugin:install ชื่อปลั๊กอิน` เพื่อติดตั้งให้เสร็จสมบูรณ์

ตัวอย่าง หากไฟล์ที่ดาวน์โหลดชื่อ ai.zip ให้แกะไปที่ `{โปรเจกต์หลัก}/plugin/ai` แล้วรัน `php webman app-plugin:install ai` เพื่อติดตั้งให้เสร็จสมบูรณ์


# การถอนการติดตั้ง

การถอนการติดตั้งปลั๊กอินแอปพลิเคชันก็มี 2 วิธีเช่นกัน:

## ถอนการติดตั้งจากตลาดปลั๊กอิน
เข้า [แผงจัดการอย่างเป็นทางการ webman-admin](https://www.workerman.net/plugin/82) ไปที่หน้าปลั๊กอินแอปพลิเคชัน แล้วกดปุ่มถอนการติดตั้งเพื่อถอนปลั๊กอินที่ต้องการ

## ถอนการติดตั้งจากแพ็กเกจซอร์สโค้ด
รัน `php webman app-plugin:uninstall ชื่อปลั๊กอิน` เพื่อถอนการติดตั้งให้เสร็จ จากนั้นลบโฟลเดอร์ปลั๊กอินที่เกี่ยวข้องใน `{โปรเจกต์หลัก}/plugin/` ด้วยตนเอง
