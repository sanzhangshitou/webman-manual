# ไฟล์กำหนดค่าเส้นทาง
ไฟล์กำหนดค่าเส้นทางของปลั๊กอินอยู่ที่ `plugin/ชื่อปลั๊กอิน/config/route.php`

## เส้นทางเริ่มต้น
เส้นทาง URL ของแอปพลิเคชันปลั๊กอินทั้งหมดเริ่มต้นด้วย `/app` เช่น `plugin\foo\app\controller\UserController` มี URL เป็น `http://127.0.0.1:8787/app/foo/user`

## ปิดการใช้งานเส้นทางเริ่มต้น
หากต้องการปิดเส้นทางเริ่มต้นของแอปพลิเคชันปลั๊กอิน ให้เพิ่มในไฟล์กำหนดค่าเส้นทางดังนี้:
```php
Route::disableDefaultRoute('foo');
```

## จัดการ callback สำหรับ 404
หากต้องการกำหนด fallback สำหรับแอปพลิเคชันปลั๊กอิน ต้องส่งชื่อปลั๊กอินเป็นพารามิเตอร์ตัวที่สอง ตัวอย่าง:
```php
Route::fallback(function(){
    return redirect('/');
}, 'foo');
```
