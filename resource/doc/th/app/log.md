# บันทึก
การใช้งานคลาสบันทึกมีความคล้ายคลึงกับการใช้งานฐานข้อมูล
```php
use support\Log;
Log::channel('plugin.admin.default')->info('test');
```

หากต้องการนำการตั้งค่าบันทึกของโปรเจกต์หลักมาใช้ซ้ำ ให้ใช้โดยตรงดังนี้
```php
use support\Log;
Log::info('เนื้อหาบันทึก');
// สมมติว่าโปรเจกต์หลักมีการตั้งค่าบันทึกชื่อ test
Log::channel('test')->info('เนื้อหาบันทึก');
```
