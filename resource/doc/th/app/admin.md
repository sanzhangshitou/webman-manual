# การเชื่อมต่อ webman-admin

[webman-admin](https://www.workerman.net/plugin/82) เป็นแผงจัดการแบคเอนด์อย่างเป็นทางการที่พัฒนาโดย webman รองรับการตั้งค่าสิทธิ์ การสร้างเมนู (CRUD แบบคลิกเดียว) และมาพร้อมตารางผู้ใช้ เพื่อความสะดวกในการแชร์ข้อมูลผู้ใช้ระหว่างปลั๊กอินแอปพลิเคชันต่างๆ

## ประโยชน์ของการเชื่อมต่อ webman-admin

* มีทางเข้าแบคเอนด์แบบรวมสำหรับปลั๊กอินแอปพลิเคชันต่างๆ เพื่อเพิ่มประสบการณ์ผู้ใช้
* ลดเวลาพัฒนาแบคเอนด์ของปลั๊กอินแอปพลิเคชัน และลดต้นทุนการบำรุงรักษา
* แผงจัดการมาพร้อมฟังก์ชันจัดการปลั๊กอิน เพื่อความสะดวกในการค้นหาและติดตั้งปลั๊กอินแบบรวมศูนย์
