# การบีบอัด

เช่น การบีบอัดปลั๊กอินแอปพลิเคชัน foo

* ตั้งค่าเลขเวอร์ชันใน `plugin/foo/config/app.php` (**สำคัญ**)
* ลบไฟล์ที่ไม่ต้องการบีบอัดใน `plugin/foo` โดยเฉพาะไฟล์ชั่วคราวทดสอบการอัปโหลดที่อยู่ใน `plugin/foo/public`
* หากโปรเจกต์ของคุณประกอบด้วยการสร้างตารางฐานข้อมูลและการดำเนินการที่คล้ายกัน ให้ตั้งค่า `plugin/foo/install.sql` อย่างถูกต้อง ดู [ส่วนฐานข้อมูล](database.md)
* หากโปรเจกต์ของคุณมีการตั้งค่าฐานข้อมูลและ Redis เป็นของตัวเอง ให้ลบการตั้งค่าเหล่านี้ก่อน การตั้งค่าเหล่านี้ควรถูกเรียกใช้ผ่านโปรแกรมช่วยการติดตั้งเมื่อเข้าถึงแอปพลิเคชันครั้งแรก (คุณต้องดำเนินการเอง) เพื่อให้ผู้ดูแลระบบกรอกและสร้างด้วยตนเอง
* หากโปรเจกต์ของคุณมีเมนู backend ของ webman admin ให้ตั้งค่า `plugin/foo/config/menu.php` เพื่อให้เมนูเหล่านี้ถูกตั้งค่าอัตโนมัติเมื่อติดตั้งปลั๊กอิน ดู [webman-admin นำเข้าเมนู](https://www.workerman.net/doc/webman-admin/app-development/menu.html)
* คืนค่าไฟล์อื่น ๆ ที่ต้องการคืนค่าให้เหมือนเดิม
* เมื่อทำขั้นตอนข้างต้นเสร็จแล้ว เข้าไปที่ `{โปรเจกต์หลัก}/plugin/`
* ผู้ใช้ Linux: ใช้คำสั่ง `zip -r foo.zip foo` เพื่อสร้าง foo.zip
* ผู้ใช้ Windows: คลิกขวาที่โฟลเดอร์ foo แล้วเลือก "บีบอัดเป็นไฟล์ ZIP" เพื่อสร้าง foo.zip

**foo.zip คือไฟล์ที่บีบอัดแล้ว ดูบทถัดไป [เผยแพร่ปลั๊กอิน](publish.md)**
