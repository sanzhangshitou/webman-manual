# ปลั๊กอินพื้นฐาน

ปลั๊กอินพื้นฐานโดยทั่วไปเป็นคอมโพเนนต์ที่ใช้ร่วมกัน โดยทั่วไปติดตั้งผ่าน Composer โค้ดจะอยู่ในโฟลเดอร์ vendor เมื่อติดตั้ง สามารถคัดลอกการกำหนดค่าแบบกำหนดเอง (เช่น middleware, process, route ฯลฯ) ไปยังโฟลเดอร์ `{โปรเจคหลัก}config/plugin` โดยอัตโนมัติ webman จะรับรู้การกำหนดค่าในโฟลเดอร์นี้โดยอัตโนมัติ และรวมเข้ากับการกำหนดค่าหลัก จึงทำให้ปลั๊กอินสามารถเข้ามามีส่วนร่วมในทุกขั้นตอนของวงจรชีวิต webman ได้

ข้อมูลเพิ่มเติม โปรดดูที่ [การสร้างปลั๊กอินพื้นฐาน](create.md)
