# Руководство по обновлению 2.1

## Возможности и изменения

* Требуется `PHP>=8.1`

* Зависимость от `workerman/workerman:^5.1`, поддержка корутин. См. [webman Coroutine](../coroutine/coroutine.md)

* Добавлен пул соединений для `webman/redis:^2.1` `webman/cache:^2.1` `webman/database:^2.1` `webman/think-orm:^2.1` `webman/think-cache:^2.1`, автоматическая поддержка среды с корутинами и без.

**Перед обновлением сделайте резервную копию и выполните следующие команды**
```
composer config -g --unset repos.packagist
composer require -W workerman/workerman:~5.1 workerman/webman-framework:~2.1
```

**Обновление базы данных**

При использовании следующих компонентов выполните соответствующие команды:

* `illuminate/database` Laravel
```
composer require -W webman/database:~2.1
```

* `illuminate/redis` Laravel
```
composer require -W webman/redis:~2.1
```

* `illuminate/cache` Laravel
```
composer require -W webman/cache:~2.1
```

* `topthink/think-orm` ThinkPHP
```
composer require -W webman/think-orm:~2.1
```

* `topthink/think-cache` ThinkPHP
```
composer require -W webman/think-cache:~2.1
```

* webman/redis-queue
```
composer require -W webman/redis-queue:~2.1
```

* webman/log
```
composer require -W webman/log:~2.1
```
