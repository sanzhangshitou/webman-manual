# Установка

Установка плагинов приложений возможна двумя способами:

## Установка из магазина плагинов
Откройте [официальную панель управления webman-admin](https://www.workerman.net/plugin/82), перейдите на страницу плагинов приложений и нажмите кнопку установки, чтобы установить нужный плагин.

## Установка из исходного пакета
Скачайте архив плагина из магазина приложений, распакуйте и загрузите распакованную папку в `{основной проект}/plugin/` (если папки plugin нет — создайте её вручную). Выполните `php webman app-plugin:install название_плагина` для завершения установки.

Например, если скачанный архив называется ai.zip, распакуйте его в `{основной проект}/plugin/ai` и выполните `php webman app-plugin:install ai` для завершения установки.


# Удаление

Удаление плагинов приложений также возможно двумя способами:

## Удаление из магазина плагинов
Откройте [официальную панель управления webman-admin](https://www.workerman.net/plugin/82), перейдите на страницу плагинов приложений и нажмите кнопку удаления, чтобы удалить нужный плагин.

## Удаление из исходного пакета
Выполните `php webman app-plugin:uninstall название_плагина` для завершения удаления. Затем вручную удалите соответствующую папку плагина в `{основной проект}/plugin/`.
