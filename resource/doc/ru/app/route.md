# Файл конфигурации маршрутов
Файл конфигурации маршрутов плагинов находится в `plugin/имя_плагина/config/route.php`.

## Маршрут по умолчанию
Все URL-пути приложений-плагинов начинаются с `/app`, например URL для `plugin\foo\app\controller\UserController` — `http://127.0.0.1:8787/app/foo/user`.

## Отключение маршрута по умолчанию
Чтобы отключить маршрут по умолчанию для приложения-плагина, добавьте следующее в конфигурацию маршрутов:
```php
Route::disableDefaultRoute('foo');
```

## Обработка callback при 404
Чтобы задать fallback для приложения-плагина, необходимо передать имя плагина вторым параметром. Пример:
```php
Route::fallback(function(){
    return redirect('/');
}, 'foo');
```
