# Логирование
Использование класса логирования аналогично использованию базы данных
```php
use support\Log;
Log::channel('plugin.admin.default')->info('test');
```

Если вы хотите повторно использовать конфигурацию логирования основного проекта, используйте напрямую:
```php
use support\Log;
Log::info('Содержимое лога');
// Предположим, в основном проекте есть конфигурация логирования с именем test
Log::channel('test')->info('Содержимое лога');
```
