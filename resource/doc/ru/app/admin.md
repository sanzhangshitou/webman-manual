# Подключение webman-admin

[webman-admin](https://www.workerman.net/plugin/82) — официальная панель управления, разработанная webman. Поддерживает настройку прав доступа, генерацию меню (CRUD в один клик), поставляется с таблицей пользователей для удобного обмена данными между плагинами приложений.

## Преимущества подключения webman-admin

* Обеспечивает единый вход в панель для различных плагинов приложений, улучшая пользовательский опыт
* Сокращает время разработки панели для плагинов и снижает затраты на обслуживание
* Панель включает функционал управления плагинами для централизованного поиска и установки
