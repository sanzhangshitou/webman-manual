# Представление

Файл конфигурации представлений для плагинов приложения находится в `plugin/{идентификатор_плагина}/config/view.php`. Использование представлений в плагинах приложения обеспечивает такой же опыт разработки, как и в обычных проектах webman. См. [Представление](../view.md) для получения дополнительной информации.
