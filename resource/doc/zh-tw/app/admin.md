# 接入 webman-admin

[webman-admin](https://www.workerman.net/plugin/82) 是 webman 官方開發的管理後台。支援權限設定、選單生成（一鍵 CRUD），自帶使用者資料表，方便各應用程式外掛共用使用者資料。

## 接入 webman-admin 好處

* 為各應用程式外掛提供統一的後台入口，提升使用者體驗
* 減少應用程式外掛後台開發時間，降低維護成本
* 管理後台自帶外掛管理功能，方便外掛的集中檢索與安裝
