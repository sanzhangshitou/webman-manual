# Phương pháp nâng cấp

`composer require workerman/webman-framework ^1.4.3 && composer require webman/console ^1.0.27 && php webman install`

> **Lưu ý**
> Proxy Composer của Alibaba Cloud đã ngừng đồng bộ dữ liệu từ kho chính thức của Composer. Hiện không thể dùng proxy Alibaba Cloud để nâng cấp lên phiên bản webman mới nhất. Vui lòng chạy lệnh sau để khôi phục việc sử dụng nguồn dữ liệu chính thức của Composer: `composer config -g --unset repos.packagist`
