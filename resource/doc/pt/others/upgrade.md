# Método de atualização

`composer require workerman/webman-framework ^1.4.3 && composer require webman/console ^1.0.27 && php webman install`

> **Observação**
> O proxy Composer da Alibaba Cloud parou de sincronizar dados do repositório oficial do Composer. Não é possível atualizar para a versão mais recente do webman usando o proxy da Alibaba Cloud. Execute o comando a seguir para voltar a usar a fonte de dados oficial do Composer: `composer config -g --unset repos.packagist`
