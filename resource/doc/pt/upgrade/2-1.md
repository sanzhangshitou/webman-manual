# Guia de atualização 2.1

## Funcionalidades e alterações

* Requer `PHP>=8.1`

* Depende de `workerman/workerman:^5.1`, suporte a coroutines. Consulte [webman Coroutine](../coroutine/coroutine.md)

* Adicionado pool de conexões para `webman/redis:^2.1` `webman/cache:^2.1` `webman/database:^2.1` `webman/think-orm:^2.1` `webman/think-cache:^2.1`, suporte automático para ambientes com e sem coroutines.

**Faça backup antes de atualizar e execute os seguintes comandos**
```
composer config -g --unset repos.packagist
composer require -W workerman/workerman:~5.1 workerman/webman-framework:~2.1
```

**Atualização do banco de dados**

Se usar os seguintes componentes, execute os comandos correspondentes:

* `illuminate/database` do Laravel
```
composer require -W webman/database:~2.1
```

* `illuminate/redis` do Laravel
```
composer require -W webman/redis:~2.1
```

* `illuminate/cache` do Laravel
```
composer require -W webman/cache:~2.1
```

* `topthink/think-orm` do ThinkPHP
```
composer require -W webman/think-orm:~2.1
```

* `topthink/think-cache` do ThinkPHP
```
composer require -W webman/think-cache:~2.1
```

* webman/redis-queue
```
composer require -W webman/redis-queue:~2.1
```

* webman/log
```
composer require -W webman/log:~2.1
```
